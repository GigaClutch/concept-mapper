Concept Mapper · review queue

You don't need to be a philosophy expert. An independent checker has already read each item and rated it — ✓ looks right, ✗ looks wrong, or needs your eye — with a plain-English reason. Green items backed by a real Stanford quote are safe to accept in bulk (use the ✨ button). For the rest, read the one-line reason and decide. The checker leans cautious, so a "looks wrong" is a prompt to look, not a verdict — trust the quote and the reason. Choices save in this browser; Export decisions downloads a file to save as data/review_decisions.json, which the pipeline applies.
